# IPL Data Analysis

Exploratory Data Analysis (EDA) of IPL matches from 2008–2024 using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## Project Objectives
- Clean and preprocess IPL match data
- Analyze team performance
- Investigate home-ground advantage
- Study toss impact on match outcomes
- Identify players with the most Player of the Match awards

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Visualizations Used
- Bar Plots
- Line Plots
- Pie Charts

## Analyses Performed
- Missing value handling
- Team name standardization
- Home vs Away win percentage analysis
- Toss winner vs match winner analysis
- Top 10 Player of the Match award winners

## Dataset
IPL Matches Dataset (2008–2024)
